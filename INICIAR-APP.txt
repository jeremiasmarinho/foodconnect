🐛 SOLUÇÃO DE PROBLEMAS

Porta ocupada?
    npx kill-port 3001    # Backend
    npx kill-port 8081    # Frontend

Cache corrompido?
    cd frontend
    rm -rf node_modules .expo
    npm install

Banco de dados corrompido?
    cd backend
    npx prisma migrate reset
    npm run db:seed
